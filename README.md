# ether-ai-openai

Adapter de Ether para OpenAI implementado con `java.net.http.HttpClient` y `ether-json`, exponiendo una API pública basada en `ether-ai-core`.

## Maven Dependency

```xml
<dependency>
    <groupId>dev.rafex.ether.ai</groupId>
    <artifactId>ether-ai-openai</artifactId>
    <version>8.0.0-SNAPSHOT</version>
</dependency>
```
